# PATH ZERO

A story-driven mathematics adventure game built as a Class 8 Mathematics Project.

## About

PATH ZERO is a 2D adventure game that combines exploration, puzzles, and mathematics. Players journey through the mysterious Whispering Forest, solving challenges based on Rational Numbers and Square Roots to progress through the story.

The game is inspired by the idea that the easiest path is not always the best path. Success comes through patience, logical thinking, and making good decisions.

## Features

- Story-driven gameplay
- Mathematical puzzles and challenges
- Rational Numbers and Square Roots integration
- Multiple levels with increasing difficulty
- Pixel-art inspired visuals
- Interactive dialogues and quests
- Educational and entertaining gameplay

## Learning Objectives

This project helps players:

- Practice Rational Numbers
- Understand Square Roots
- Improve logical thinking
- Develop problem-solving skills
- Learn through interactive gameplay

## Story

A traveler enters the Whispering Forest in search of a way forward. Along the journey, mathematical challenges stand between the player and their goal. Every puzzle solved reveals a little more about the world and teaches an important lesson: shortcuts are not always the smartest choice.
